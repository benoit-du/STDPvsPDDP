This code can be used to compare the evolution of a Kuramoto network with spike-timing-dependent plasticity (STDP) 
to the evolution of a Kuramoto network with phase-difference-dependent plasticity (PDDP) and event-based PDDP (ebPDDP).
See Duchet et al., Neural Comput., 2023 for more details.   
*************************************************
Start by running 'test_script.m' (should take around 5min with default params)
